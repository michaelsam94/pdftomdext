# Change Log

All notable changes to this project will be documented in this file.

## [0.0.4]
- Improved coordinate-based PDF text layout parsing for better table row/column detection.
- Version bump and repackaged release.

## [0.0.3]
- Improved PDF table-like block conversion to Markdown table output.
- Version bump and repackaged release.

## [0.0.2]
- Confirmed extension branding name as `PdfToMd`.
- Version bump and repackaged release.

## [0.0.1]
- Initial release of PdfToMd.
- Added Explorer right-click conversion support for `.pdf` files.
- Added configurable output folder, overwrite behavior, and auto-open support.
